Our services

Business Convergence offers IT support serices that include the technical docuemntation to support the application directly and the associated quality management systems to ensure the software remains in a state of control throughout the applications life-cycle.

Information Technology (IT) Services

Business Convergence provides analysists and technical docuemnt specialists to create the foundation of each software applicaiton.  The foundation includes gathering formal user requirements that define the intended application then providing a protocol to tst that each of the requirements is properly tested.

IT Documents include:

Risk Assessment, Qualificaiton Plan, User Requirements, Testing protocol, Test scripts, tracability matrix (each requirement maps to a test), User Acceptance Testing, Qualificaiton Report (how well did you stick to your plan), Knowledge Transfer, Training Manuals, Standard Operating Procedures (SOPs), Production Support and application trianing.

 

Quality Management Systems (QMS)

IExternal to the application there are support services that can be leveraged accross an organization.  These Polies and Procedures are docuemented to ensure a consistant life-cycle methodology to all applications.

QMS Documents include:

Configuration management, Insident Management, Channge Control, Backup and Recorery, Archiving, promotion to the Production Environment, Documentation Management, Contingency Planning (manual process), application retirement, and QMS traing on Learning Management System (LMS).
